## 📋 License Guide (Simple)

### 🟢 **CC0 License** (Best! Most sites use this)
- Use for **anything** — personal projects, commercial products, websites
- You can edit, modify, sell — do whatever you want
- You **don't** need to give credit
- Found on: Unsplash, Pexels, Pixabay

### 🟡 **CC BY License** (Free, but give credit)
- Use for **anything** — commercial, personal, edit, sell
- You **must** credit the creator
- Found on: Freesound, some Flickr images

### 🟠 **CC BY-SA License** (Share-alike)
- Use for **anything**, but if you share it, use the **same license**
- You **must** credit the creator
- Found on: Wikipedia images

### 🔴 **Site-Specific Licenses**
- Some sites (Freepik, Icons8) have their own rules
- Usually requires attribution or limits downloads per day
- **Always check** the website's license page

> 💡 **Rule of thumb:** When unsure, **give credit**. It costs nothing and keeps everyone happy.
