# 🎨 Free Color Tools & Palettes

Websites that help you choose, create, and test colors. Perfect for branding, website design, social media, art, and presentations.

---

## 1. ⭐ Coolors

**🔗 https://coolors.co**

A color palette generator. Press the spacebar and get beautiful color combinations instantly.

- **Best for:** Creating color schemes for any project
- **How to use:**
  1. Go to the website
  2. Press **spacebar** to generate random palettes
  3. **Lock** colors you like by clicking the lock icon
  4. Keep pressing spacebar until you're happy
  5. Export as CSS, SVG, PNG, PDF
- **Also:** Color blindness simulator, browse trending palettes

---

## 2. ⭐ ColorHunt

**🔗 https://colorhunt.co**

A curated collection of beautiful color palettes. Simple and clean.

- **Best for:** Color inspiration, finding ready-made palettes
- **How to use:**
  1. Browse the palettes on the homepage
  2. Click one you like to see the colors
  3. Copy the hex codes and use them
- **Updated daily** with new palettes

---

## 3. ⭐ Adobe Color

**🔗 https://color.adobe.com**

Adobe's professional color wheel tool. Create color schemes using color theory rules.

- **Best for:** Professional color design, color theory, theme extraction
- **How to use:**
  1. Choose a color rule (analogous, complementary, triad, etc.)
  2. Move the color wheel to adjust
  3. Or upload an image to extract colors from it
  4. Export to Adobe products
- ⭐ Industry standard used by professional designers

---

## 4. ⭐ ColorHexa

**🔗 https://www.colorhexa.com**

A color encyclopedia. Enter any color and get EVERYTHING about it.

- **Shows:** Hex, RGB, HSL, CMYK, HSV values
- **Also:** Similar colors, color blindness simulation, gradient generator
- **Best for:** Understanding colors, conversions, color details

---

## 5. ⭐ MyColor.Space

**🔗 https://mycolor.space**

A gradient generator. Enter one color and get beautiful gradient combinations.

- **Best for:** Creating gradients for websites, backgrounds, buttons, cards
- **Special:** Preview on different backgrounds, copy CSS code directly

---

## 💡 Tips for Beginners

- Start with **Coolors** — fastest way to get a color palette
- Use **ColorHunt** when you want pre-made, beautiful palettes
- Use **Adobe Color** when you need professional color theory
- Keep it simple: **2-3 colors maximum** for most projects
- Always test your colors on both **light and dark** backgrounds
- Write down the **hex codes** (#ffffff) to use later
- Most design tools accept hex codes directly
