1.  ⭐  COOLORS

     🔗  https://coolors.co

     What it is:
     A color palette generator. Press the spacebar and get
     beautiful color combinations instantly.

     Why use it:
     •  Generate palettes with one click (spacebar)
     •  Lock colors you like and regenerate the rest
     •  Export to CSS, SVG, PNG, PDF, Adobe Swatch
     •  Browse trending palettes from other users
     •  Color blindness simulator included

     Best for:
     Creating color schemes for any project

     How to use:
     1. Go to the website
     2. Press spacebar to generate random palettes
     3. Lock colors you like by clicking the lock icon
     4. Keep pressing spacebar until you're happy
     5. Export and use in your project


2.  ⭐  COLORHUNT

     🔗  https://colorhunt.co

     What it is:
     A curated collection of beautiful color palettes. Simple and clean.

     Why use it:
     •  Hand-picked palettes — every one is beautiful
     •  Updated daily with new palettes
     •  Copy hex codes with one click
     •  No ads, no distractions — just colors

     Best for:
     Color inspiration, finding ready-made palettes

     How to use:
     1. Browse the palettes on the homepage
     2. Click one you like to see the colors
     3. Copy the hex codes and use them


3.  ⭐  ADOBE COLOR

     🔗  https://color.adobe.com

     What it is:
     Adobe's professional color wheel tool. Create color schemes
     using color theory rules.

     Why use it:
     •  Color wheel with rules (analogous, monochromatic, triad, etc.)
     •  Extract color themes from uploaded images
     •  Industry standard — used by professional designers
     •  Browse community-created themes
     •  Export to Adobe products

     Best for:
     Professional color design, color theory, theme extraction

     How to use:
     1. Choose a color rule (analogous, complementary, etc.)
     2. Move the color wheel to adjust
     3. Or upload an image to extract colors from it
     4. Save and export your theme


4.  ⭐  COLORHEXA

     🔗  https://www.colorhexa.com

     What it is:
     A color encyclopedia. Enter any color and get EVERYTHING
     about it.

     Why use it:
     •  Shows hex, RGB, HSL, CMYK, HSV values
     •  Shows similar colors and alternatives
     •  Color blindness simulation
     •  Gradient generator
     •  Convert between color formats

     Best for:
     Understanding colors, conversions, color details

     How to use:
     Just type a color name or hex code and press enter


5.  ⭐  MYCOLOR.SPACE

     🔗  https://mycolor.space

     What it is:
     A gradient generator. Enter one color and get beautiful
     gradient combinations.

     Why use it:
     •  Generates beautiful gradients from any color
     •  Preview on different backgrounds (dark/light)
     •  Copy CSS code directly
     •  Multiple gradient style options

     Best for:
     Creating gradients for websites, backgrounds, buttons, cards


💡  TIPS FOR BEGINNERS:

     •  Start with Coolors — fastest way to get a color palette
     •  Use ColorHunt when you want pre-made, beautiful palettes
     •  Use Adobe Color when you need professional color theory
     •  Keep it simple: 2-3 colors maximum for most projects
     •  Always test your colors on both light and dark backgrounds
     •  Write down the hex codes (#ffffff) to use later
     •  Most design tools accept hex codes directly
