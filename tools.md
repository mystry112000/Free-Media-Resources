# 🛠️ Free Design Tools

Software (mostly online) for creating and editing graphics. Perfect for photo editing, graphic design, UI/UX design, and image optimization.

---

## 1. ⭐ Canva

**🔗 https://www.canva.com**

An online design tool for non-designers. Drag, drop, and create beautiful graphics in minutes.

- **Best for:** Social media graphics, presentations, posters, thumbnails, flyers
- **Free tier includes:** 250,000+ templates, 5GB cloud storage, hundreds of fonts
- **How to use:**
  1. Create a free account
  2. Pick a template or start from blank
  3. Drag and drop elements, change text and colors
  4. Download or share directly

---

## 2. ⭐ Photopea

**🔗 https://www.photopea.com**

A free online alternative to Adobe Photoshop. Runs in your browser. No installation needed.

- **Best for:** Photo editing, PSD files, advanced graphic design, mockups
- **Supports:** PSD, AI, Sketch, XD, Figma files
- **Features:** Layers, masks, filters, text tools — everything Photoshop has
- **No account needed** | Minimal ads

---

## 3. ⭐ Figma

**🔗 https://www.figma.com**

A professional UI/UX design tool. Industry standard for website and app design.

- **Best for:** Website design, app design, UI/UX, prototyping, wireframing
- **Free tier includes:** Unlimited files, 3 projects, version history
- **Special:** Real-time collaboration (like Google Docs for design)
- **How to use:**
  1. Create a free account
  2. Start with a template or blank canvas
  3. Use tools (frames, shapes, text, images)
  4. Share with others for feedback

---

## 4. ⭐ Remove.bg

**🔗 https://www.remove.bg**

AI-powered tool that removes backgrounds from images automatically.

- **Best for:** Product photos, profile pictures, thumbnails, e-commerce images
- **Free tier:** 50 images per month
- **How to use:** Upload image → background gone in seconds

---

## 5. ⭐ TinyPNG

**🔗 https://tinypng.com**

Image compression tool. Makes image files smaller without losing visible quality.

- **Best for:** Website images, social media images, email attachments
- **Reduces file size by:** 50-80%
- **Free tier:** Compress images up to 5MB each. No account needed.

---

## 6. ⭐ SVGO

**🔗 https://svgomg.net**

Tool that optimizes SVG files. Makes them smaller and cleaner.

- **Best for:** Optimizing SVG icons and illustrations for web use
- **How to use:** Just drag and drop your SVG file

---

## 💡 Tips for Beginners

- **Canva** is the easiest to start with — use it for social media
- **Photopea** when you need Photoshop features without paying
- **Figma** when you're designing websites or apps
- Always use **TinyPNG** before uploading images to a website
- **Remove.bg** is a lifesaver for product photos
- Most of these tools work in **browser** — nothing to install
